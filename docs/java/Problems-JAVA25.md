문제5. 왼쪽 순서도에서 잘못된 부분은 반복문이 false일때 반복된다는 것이다. 찾아서
true가 반복되게 고친 다음 의사코드를 만들어 보자.
```
초코 분말이 섞였는가? 를 초코 분말이 섞이지 않았는가? 로 변경 후 해당 yes,no를 반전
```
```
주문이 들어온다.
계산을 한다.
if(설탕을 넣는가?){
    설탕을 넣는다.
} else {
    시럽을 넣는다.
}
얼음을 넣는다.
while(초코 분말이 섞이지 않았는가?){
    저어준다.
}
주문번호를 부른다.
```
