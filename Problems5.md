1. 프로그램의 목적은 무엇인가? 
```
현실 세계의 데이터를 컴퓨터에 넣어 사용자가 필요로 할 때 데이터를 가공하여 원하는 결과를 보여주는 것
```

2. 자료형이란? 
```
현실 세계의 다양한 데이터를 효율적으로 저장하고 처리하기 위해 프로그램에서 미리 정의된 데이터의 형식
```

3. 자료형의 종류와 어떤 자료를 담는지 기술하시오.

```
불리언 boolean
문자열 String
```
